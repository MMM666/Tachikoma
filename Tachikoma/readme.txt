蜘蛛のモデルをタチコマに置換えるMOD Tachikoma 1.6.2 Rev2


蜘蛛のモデルを、おこたさん作成littleMaidMob用マルチモデル「タチコマモデル」に置き換えるMODです。
フラグ回収のために作ったので適当。



利用条件
	・モデル作者であるおこたさんに感謝の祈りを捧げてください。
	・利用条件はおこたさんのモデルと同等になっています。
	・動画等での使用、改造、転載すきにしてもよいのよ？
	・ただし、商用利用は除く。
	・あと、いかなる意味でも作者は責任をとりませぬ。



使い方
	要Modloader。
	要MMMLib Rev5。
	AudioModが有れば鳴き声に好きな音声を指定できます。
	%appdata%/.mincraft/versions/1.6.2ML/modsにZIPのまま放りこめば動くはずです。
	%appdata%/.mincraft/config/mod_Tachikoma.cfgが設定ファイルです、適当にどうぞ。



効能
	・蜘蛛のモデルが「タチコマモデル」互換に置き換わります。
	・洞窟蜘蛛のモデルは変わりません。
	・敵対していない蜘蛛にはRIDE-ONできます。
	・RIDE-ONした蜘蛛は別に操作ができるわけではありません。
	・cfgで設定すれば蜘蛛の鳴き声を変える事が出来ます。
	・littleMaidMobを使用している時にこちらのMODを同時に入れておくとメイドさんのマルチモデルにタチコマが追加されます。



注意
	・バックアップはこまめに取りましょう。
	・前バージョンと違って単独動作します。
	・cfgで蜘蛛の大きさを変えられます。（w1.0 x h1.7）
	・cfgで光学迷彩を有効にすると、かなり鬼畜な蜘蛛がスポーンするようになります。
	・メイドさんのマルチモデルとしてのみ使用したい場合はcfgにて設定を行なって下さい。
